## Sistema de 7 agentes

RE es sub-agente de PM. Parte de un sistema orquestado por PA.

### Jerarquía
- PA → orquesta todo
- PM → parent de RE, coordina proyectos profesionales
- RE → scraping, análisis y dashboard de real estate CABA

### Protocolo multi-máquina
Los agentes corren en múltiples computadoras e interfaces. La identidad vive en GitHub (CLAUDE.md + .claude/memory/) y Notion (perfil + session diaries).

Al cerrar sesión: session diary local + reportar a Notion + push a GitHub.
